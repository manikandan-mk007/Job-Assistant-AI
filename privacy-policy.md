# Privacy Policy for AI Job Assistant

Last updated: April 2026

AI Job Assistant is a browser extension designed to help users analyze job postings and improve their applications.

## Information We Collect

- Job details from supported websites (e.g., LinkedIn)
- User profile data (skills, role, education)
- Resume data (if uploaded)

## How We Use Information

We use this data to:
- Calculate job match score
- Identify missing skills
- Suggest keywords
- Generate cover letters
- Save job data

## Data Storage

- Data may be stored securely in backend systems
- We do NOT sell or share user data

## User Control

- Users choose what data to provide
- Users can stop using the extension anytime

## Security

We take reasonable measures to protect data but cannot guarantee complete security.
